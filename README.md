# 사용법
1. 리눅스 환경에 Makefile과 myshell.c 파일을 다운받아 make 명령어를 입력한다.
2. ./myshell 명령어를 입력한다.
3. cmdlist를 입력하여 구현된 커맨드들을 확인한다.
4. cd '파일이름' 을 입력하면 해당 파일로 이동한다.
5. cd를 입력하면 루트 디렉토리로 이동한다.
6. hello를 입력하면 hello world가 출력된다.
7. pwd를 입력하면 현재 작업경로가 출력된다.
8. exit를 입력하면 종료된다.
